Algoritma Menghitung Rata-rata dari N Bilangan

Di banyak aplikasi komputasi, kita sering perlu menghitung rata-rata dari sejumlah data. Misalnya, dalam sistem penilaian siswa, kita mungkin perlu menghitung rata-rata nilai dari beberapa mata pelajaran. Anda diberikan sejumlah angka, dan tugas Anda adalah menghitung rata-rata nilai tersebut.

Mari kita asumsikan Anda diberi 5 nilai: 85, 90, 78, 88, dan 92. Anda diminta untuk menghitung rata-rata dari nilai-nilai tersebut. Ini adalah contoh yang sangat umum dalam dunia pendidikan atau bahkan analisis data.

Berikut langkah-langkah dalam bentuk deskriptif, flowchart, pseudocode dan juga coding c++.

Algoritma Deskriptif

Deskripsi Masalah:

Buatlah algoritma untuk menentukan apakah sebuah bilangan yang diberikan adalah genap atau ganjil.

Notasi Algoritma Deskriptif:

1. Mulai

2. Input jumlah bilangan N yang akan dihitung rata-ratanya

3. Inisialisasi sum = 0 (untuk menyimpan jumlah angka)

4. Untuk setiap bilangan yang dimasukkan (mulai dari 1 hingga jumlah bilangan):

- Input angka

- Tambahkan angka ke dalam sum

5. Hitung rata-rata dengan rumus: average = sum / N

6. Tampilkan hasil rata-rata

7. Selesai

Berikut adalah flowchart untuk menghitung Rata-rata dari N Bilangan

Pseudocode

            
Mulai
	Deklarasikan N, angka, sum sebagai int 
	Deklarasikan average sebagai double
    Input jumlah bilangan N
    Inisialisasi sum = 0
    Untuk i dari 1 sampai N
        Input angka ke-i
        Jumlahkan angka ke-i ke dalam sum
    Hitung rata-rata: average = sum / N
    Tampilkan average
Selesai

Coding C++

            
#include <iostream>
using namespace std;

int main() {
    int N, angka, sum = 0;
    double average;

    cout << "Masukkan jumlah bilangan (N): ";
    cin >> N;

    for (int i = 1; i <= N; i++) {
        cout << "Masukkan angka ke-" << i << ": ";
        cin >> angka;
        sum += angka;
    }

    average = sum / N;

    cout << "Rata-rata dari " << N << " bilangan adalah: " << average << endl;

    return 0;
}
